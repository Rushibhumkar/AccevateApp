Accevate React Native Application
------------------------------------

This is a React Native mobile application built as part of an assignment.
The app includes Login, OTP Verification, and a Dashboard with dynamic UI behavior based on API responses.

How to Run the Application
-----------------------------

Make sure the following are installed on your system:-

* Node.js (v18+ recommended)
* npm
* React Native CLI
* Android Studio (with emulator) or a physical Android device
* Java JDK 17

Steps to Run
--------------

1. Clone the repository --->>>
  
git clone https://github.com/AnubhaAnu/AccevateApp.git
cd AccevateApp


2. Install dependencies

npm install


3. Start Metro bundler

npx react-native start


4. Run the app on Android

npx react-native run-android


Ensure an Android emulator is running or a physical device is connected with USB debugging enabled.
